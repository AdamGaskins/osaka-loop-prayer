# Developer Documentation

## Updating station information

1. Download the above spreadsheet as an XLSX file
1. Run the following command:
   `./scripts/data_convert.ts to_yaml "~/Downloads/Osaka Loop Prayer Content.xlsx"`

## Developing

```
npm install
npm run dev
```

## Building

```
npm run build
```
